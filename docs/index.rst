================================
The New Hotness Messaging Schema
================================

This is a documentation for The New Hotness Fedora Messaging Schema.

It's goal is to provide unified messaging schema for
`The New Hotness project <https://github.com/fedora-infra/the-new-hotness>`_. 
Schema is defined for `fedora messaging
<https://fedora-messaging.readthedocs.io/en/stable/>`_.

:Github page: https://github.com/fedora-infra/the-new-hotness-messages
